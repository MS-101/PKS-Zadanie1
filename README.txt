Inštrukcie
==========================================================================================================================
1. Súbor ktorý chcete analyzovať vložte do priečinku Input.
2. Spustite program zo súboru "main.cpy".
3. Do spusteného programu po vyzvaní zadajte názov požadovaného súboru s príponou.
4. Do spusteného programu po vyzvaní zadajte kód výstupného súboru(uvedené nižšie).
5. Vygenerovaný výstupný súbor môžete nájsť a skontrolovať v adresári Output.
